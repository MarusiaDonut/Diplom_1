# Дипломная работа "Задание №1 - юнит-тесты"

Необходимо написать юнит-тесты для сайта Stellar Burgers.

## Технологии:

| Технология   | Версия |
|--------------|--------|
| Java     |     11     |
| JUnit |       4.13.2  |
| Maven  |    3.9.0     |
| Jacoco      |   0.8.7      |
| Mockito      |    3.12.4    |

## Запуск тестов:
`mvn clean test`
